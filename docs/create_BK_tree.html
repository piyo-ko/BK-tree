Creating a new BK-tree
BK 木の作成

  1. Input values (character strings) in a way in which each one line includes one value. / 1 行に 1 個ずつ値 (文字列) を入力してください。
  2. Tap (or click) this button / このボタンをタップ (ないしクリック) してください:
  3. A BK-tree will be generated. / BK 木が作成されます。
  4. The BK-tree in JSON format will be indicated below. / BK 木が JSON 形式で表示されます。
  5. The BK-tree will also be visualized below. / BK 木は描画もされます。
