Visualizing a BK-tree and downloading its image file
BK 木の視覚化と画像ファイルのダウンロード

The sample BK-tree of the test page is visualized here. The resultant image (in PNG format) will be downloaded. Note that draw_BK_tree() function may fail to work well when applied to a very huge BK-tree.

サンプルの BK 木 (テストページを参照) の絵を描きます。その結果できた画像 (PNG ファイル) をダウンロードします。なお、巨大な木に対して draw_BK_tree() 関数を適用した場合は、うまく動作しないことがあります。
